\# 鼠标三维重建与地形跟随系统



> 软件工程大二学生 | 三维视觉入门项目 | 空间智能实践



\## 📌 项目简介



本项目实现了一个完整的 \*\*2D 照片 → 3D 模型 → 智能交互\*\* 的流程。从 30 张手机拍摄的鼠标照片出发，通过 COLMAP 重建出 3D 点云，再生成三角网格模型，最后实现虚拟小球在模型表面的自动地形跟随。



\*\*核心亮点\*\*：

\- ✅ 在无 GPU 环境下完成全流程

\- ✅ 从零实现射线检测和地形跟随算法

\- ✅ 可视化行走轨迹，验证算法正确性

\- ✅ 完整的项目文档和代码注释

## 深度学习地形预测模型

### 训练配置
- **网络结构**：2 → 64 → 32 → 1 (MLP)
- **损失函数**：MSE (均方误差)
- **优化器**：SGD (学习率 0.001)
- **训练轮数**：1000
- **数据量**：3,792 个采样点

### 训练结果
- **最终损失**：0.0131
- **收敛情况**：稳定下降，无明显过拟合

### 输出内容
1. `loss_curve.png` - 训练损失曲线图
2. `terrain_prediction.png` - AI预测的3D地形图
3. `terrain_model.pth` - 训练好的模型权重
